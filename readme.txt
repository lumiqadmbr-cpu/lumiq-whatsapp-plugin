Distribua leads automaticamente para sua equipe de vendas via WhatsApp com sistema inteligente de fila.

== Description ==

O LUMIQ WhatsApp transforma seu site WordPress em uma máquina de captura e distribuição de leads via WhatsApp.

**🎯 Principais Funcionalidades:**

* **Distribuição Inteligente** - Round Robin (fila circular), Aleatória ou Menor Carga
* **Múltiplas Equipes** - Organize vendedores por região, produto ou departamento
* **Botão Flutuante** - WhatsApp sempre visível no seu site
* **Formulário Customizável** - Capture nome, email, telefone e mensagem
* **Modo Direto** - Redireciona para WhatsApp sem formulário
* **Analytics Integrado** - Acompanhe conversões e performance
* **Fácil Instalação** - Configure em 3 minutos

**🚀 Como Funciona:**

1. Visitante clica no botão WhatsApp do seu site
2. Preenche formulário rápido (ou vai direto)
3. LUMIQ distribui automaticamente para o próximo vendedor disponível
4. Lead é redirecionado para WhatsApp do vendedor
5. Vendedor recebe notificação e inicia atendimento

== Installation ==

**Instalação Automática:**

1. Acesse WordPress Admin → Plugins → Adicionar Novo
2. Busque por "LUMIQ WhatsApp"
3. Clique em "Instalar Agora"
4. Ative o plugin

**Instalação Manual:**

1. Faça download do arquivo .zip
2. Acesse WordPress Admin → Plugins → Adicionar Novo → Upload
3. Selecione o arquivo .zip
4. Clique em "Instalar Agora"
5. Ative o plugin

**Configuração:**

1. Crie uma conta gratuita em [LUMIQ.io](https://lumiq.io)
2. No painel LUMIQ, vá em WordPress e gere uma API Key
3. No WordPress, vá em LUMIQ WhatsApp → Configurações
4. Cole sua API Key e clique em "Validar Chave"
5. Escolha a equipe que receberá os leads
6. Ative o botão flutuante
7. Pronto! Teste no seu site

== Frequently Asked Questions ==

= O plugin é gratuito? =

Sim! O plugin é 100% gratuito. Você precisa de uma conta LUMIQ (também gratuita com plano Starter).

= Preciso saber programar? =

Não! A instalação é simples e não requer conhecimento técnico.

= Funciona com qualquer tema WordPress? =

Sim! O LUMIQ WhatsApp é compatível com todos os temas WordPress.

= Quantos vendedores posso adicionar? =

No plano gratuito você pode ter até 5 vendedores. Planos pagos têm vendedores ilimitados.

= Posso ter múltiplas equipes? =

Sim! Você pode criar equipes por região, produto ou departamento.

= Como funciona a distribuição Round Robin? =

O sistema mantém memória da fila. Quando o vendedor #3 recebe um lead, o próximo vai para o #1, garantindo distribuição igual.

= Os dados dos leads ficam salvos? =

Sim! Todos os leads ficam salvos no painel LUMIQ com histórico completo.

= Funciona em mobile? =

Perfeitamente! O botão é responsivo e funciona em qualquer dispositivo.

= Tem suporte em português? =

Sim! Suporte completo em português via chat, email e documentação.

= Posso customizar as cores do botão? =

Sim! Você pode escolher cor, posição, texto e muito mais.

== Privacy Policy ==

LUMIQ WhatsApp coleta e processa os seguintes dados:

* Nome, telefone, email e mensagem dos visitantes (apenas quando preenchido no formulário)
* API Key do usuário (armazenada localmente no WordPress)
* Dados de uso (cliques, conversões) enviados para plataforma LUMIQ

Todos os dados são processados conforme nossa [Política de Privacidade](https://lumiq.io/privacidade) e em conformidade com a LGPD.

Não vendemos, compartilhamos ou usamos seus dados para fins publicitários.
